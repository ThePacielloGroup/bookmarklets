  • ARC Capture Snippets. You can run scripts against ARC Capture by opening the Dev Tools, going to 'Source' tab and then using the 'Snippets' section. Full details on how to use Snippets are here. The zip file contains the following scripts:
    • 1 Filter - Engagements
      • Engagements that have items ready for QA - engagements-that-can-be-QAd.js
      • Remove any engagements that are not currently active - remove-engagements-that-are-not-running.js
    • 2 Filter - Component Status DEPRECATED - now part of Capture by default
      • In Progress Only: - Filter-In-Progress-Only.js
      • Under Review Only: - Filter-Under-Review-Only.js
      • Completed Only: - Filter-Completed-Only.js
    • 3 Decrapulating Rules
      • Hide the rule descriptions (Automated)- hide-automated-rules-descriptions.js DEPRECATED - now part of Capture by default
      • Hide the rules (Automated) - hide-automated-rules-tables.js
      • Hide the rule descriptions (Manual)- hide-manual-rules-descriptions.js DEPRECATED - now part of Capture by default
      • Hide the rules (Manual) - hide-manual-rules-tables.js
    • 4 Fault finding
      • Does same as existing 'Expand All' on the Assertions list view in a component but also flags any that are missing targets (which are always required) - QA-Expand-all-assertions-and-flag-missing-targets.js DEPRECATED - now part of Capture by default (missing targets flagged automatically)
    • 5 Reporting
      • Creates an ASCII representation of how far along the line an engagement is and sorts the components into In Progress, Under Review and Completed - QA-Status-from-ARC-Capture.js
    • 6 Misc
      • View the thumbnail images as a large group of tiles (for better visibility). Works along with the filters (so you could filter items 'In Progress' then apply this) - view-ARC-components-as-gallery.js DEPRECATED - now part of Capture by default
      • NEW: Click to select parent/child relationships with assertions then get the generated text to paste into those assertions (Parent issue, Related/Child issues) - select-related-assertions.js
